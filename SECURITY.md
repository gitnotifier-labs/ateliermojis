# Security Policy

## Supported Versions

At the moment, only the latest code on `main` is actively supported with security fixes.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report vulnerabilities privately by emailing:

- opensource@gitnotifier.com

Please include:

- A description of the issue and impact.
- Reproduction steps or proof of concept.
- Any suggested remediation.

We will acknowledge receipt as quickly as possible and keep you informed during
triage and remediation.
